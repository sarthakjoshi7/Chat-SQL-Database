# Chat SQL Database

A LangChain based AI application that enables users to interact with a SQL database using natural language.

## Project Goal

This project demonstrates how Large Language Models can be integrated with SQL databases to enable natural language querying, making database interaction easier and more intuitive without requiring SQL knowledge.

## Features

- Chat with SQL database using natural language
- Supports MySQL and SQLite database
- AI powered SQL query generation
- Converts user queries into SQL automatically
- Real time responses using Open Source LLM
- Chat history support
- User interface

## Tech Stack

- Python
- LangChain
- Streamlit
- SQLAlchemy
- MySQL
- SQLite
- Llama 3.1
- Open Source LLM API

End of project.
